# Netflix Analytics Dashboard using POWER BI
### Project Title
<b>Netflix Analytics Dashboard</b>

### Project Overview
The <b>Netflix Analytics Dashboard</b> provides a comprehensive look into streaming performance, subscriber demographics, and financial metrics. This project focuses on analyzing user engagement and revenue patterns to understand what drives platform growth.

### Purpose
The purpose of this dashboard is to evaluate:<br>
•	<b>User Engagement:</b> Analyzing total watch hours and average watch time per user.<br>
•	<b>Content Popularity:</b> Identifying top-performing shows and high-demand genres.<br>
•	<b>Financial Health:</b> Tracking monthly revenue trends and average revenue per user (ARPU).<br>
•	<b>Platform Accessibility:</b> Understanding which devices users prefer for streaming.

### Tools Used
•	<b>Power BI Desktop:</b> Power BI Desktop: Used for the overall layout, design, and interactivity of the dashboard.<br>
•	<b>Power Query Editor:</b> Utilized for ETL (Extract, Transform, Load) processes, data cleaning, and transformation to ensure data accuracy.<br>
•	<b>DAX:</b> Used to create complex measures and calculated columns for custom metrics (e.g., Total Sales, Average Sales).<br>
•	<b>Design:</b> Custom dark-themed UI/UX layout optimized for executive reporting.


### Features and Highlights
• <b>Streaming KPIs:</b> High-level metrics showing 9M Total Subscribers, 569.4 Total Watch Hours, and ₹55.42M Monthly Revenue.<br>
• <b>Interactive Filtering:</b> Slicers for Country, Date, and Genre to drill down into specific data segments.<br>
• <b>Growth Tracking:</b> Trend lines showing Monthly Subscriber Growth and Monthly Revenue fluctuations throughout the year.<br>
• <b>Device Analysis:</b> A breakdown showing a near-even split between Mobile (35%), Laptop (34%), and T.V. (31%).<br>
• <b>Subscription Type:</b> A comparison showing that Movies (66%) lead over TV Shows (34%) in subscriber preference.<br>
• <b>Global Reach:</b> A geographic heatmap visualizing subscriber density across the globe.
